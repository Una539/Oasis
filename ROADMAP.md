# Oasis Flow 开发路线图

> **版本**: 0.5.1 → 按需增量演进
> **最后更新**: 2026-07-03

---

## 迭代计划

| #   | 功能                      | 目标版本 | 简述                                                                                  |
| --- | ------------------------- | -------- | ------------------------------------------------------------------------------------- |
| 1   | 归档视图 + 智能分区       | v0.4.0   | 已完成：默认显示"今天"，已完成任务自动归档，支持 Today / Upcoming / Inbox / Archived 分区切换 |
| 2   | 优先级系统                | v0.4.1   | 已完成：1-5 级优先级，通过左侧竖条宽度与颜色区分视觉权重                                      |
| 3   | 多标签系统                | v0.4.2   | 已完成：自由创建标签、预设调色板、按标签筛选                                                  |
| 4   | 截止日期增强 + 系统通知   | v0.4.3   | 已完成：逾期高亮、今日 badge、单任务提醒开关、"重新安排"交互、今日到期系统通知                  |
| 5   | 统计面板                  | v0.5.0   | 已完成：本周完成数、连续完成天数、7 天完成趋势图                                              |
| 6   | 系统通知                  | v0.6.0   | 温和桌面通知，仅在今天有到期任务且应用未聚焦时推送一次                                |
| 7   | 导出/导入                 | v0.7.1   | JSON 完整导出与导入，支持合并/替换模式                                                |
| 8   | 自然语言输入              | v0.8.0   | 输入框解析 `买牛奶 tomorrow #生活 !3` 为内容+日期+标签+优先级                         |
| 9   | 系统托盘快速输入          | v0.8.1   | 托盘迷你浮窗，直接输入回车添加                                                        |
| 10  | 应用内快捷键              | v0.8.2   | Ctrl+N 新建、Ctrl+F 搜索、Ctrl+1-4 切换分区等                                         |
| 11  | 排序 & 筛选               | v0.8.3   | 按优先级、日期、标签等维度排序和筛选                                                  |
| 12  | 拖拽排序                  | v1.0.0   | 手动调整任务顺序                                                                      |
| 13  | 子任务 / Checklist        | v1.1.0   | Todo 下展开多个子任务                                                                 |
| 14  | 主题自定义                | v1.2.0   | 用户可选择自定义强调色                                                                |
